This was a project I made with a partner Kylie Carroll. The goal was to use a project from Hyperakt as a starting point for out project that addressed an issue we wanted to talk about. We chose the Hyperakt project called ClimateWorks Reports which is an international network that works to increase public understanding of climate chaos and to reduce carbon emissions.

For our project we decided to make a meal for ourselves, a salad, and everything we used to make the meal and in this project we repurposed. With the leftover food we made different colored dyes, with the initial printed on paper iterations we were required to make for class we recycled and made new homemade paper, with the table cloth we made a jacket, and with any plastic we had we made string to sew together this jacket. Above are pictures of the book we made using this homemade, recycled paper about the dyes we made and the no waste pattern cut we used for the jacket and pictures of the jacket we made. For more about the process of this project visit medium blog post.
